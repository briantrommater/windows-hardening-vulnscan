Objective

This project demonstrates the ability to apply Windows hardening, run a basic vulnerability scan, and document security findings clearly.

Sample PowerShell Script

# disable_services.ps1
Set-Service -Name "Fax" -StartupType Disabled
Set-Service -Name "XblGameSave" -StartupType Disabled
Set-Service -Name "DiagTrack" -StartupType Disabled


Write-Output "Selected services disabled successfully."

For full documentation, see the repository contents.
